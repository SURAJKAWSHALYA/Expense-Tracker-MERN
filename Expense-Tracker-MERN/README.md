# Expense Tracker MERN Application

## Project Overview
A full-stack expense tracking system where users can manage income and expenses in a clean, professional web application.

## Purpose
This project is being built as a production-ready MERN stack application to help users track daily spending, monitor budgets, and analyze financial activity over time.

## Technology Stack
- Frontend: React.js with Vite and JavaScript
- Backend: Node.js with Express.js
- Database: MongoDB with Mongoose
- Architecture: Client-server, REST API, MVC-oriented structure, separation of concerns

## Current Development Phase
Phase 1: Project Planning & Architecture

## Future Features
- User authentication
- Expense CRUD operations
- Dashboard analytics
- Categories
- Monthly reports

## Project Structure
- client: React frontend application
- server: Express backend API
